## How to use this template

- Setup with `yarn`
- Run `node ./scripts/reset.js` to start fresh
- Run dev server with `yarn dev`
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Built with

- Next.js 13
- TailwindCSS 3.2
- Framer Motion
